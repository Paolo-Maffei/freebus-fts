Freebus Tools Software
----------------------

Ein Programm zur Konfiguration von KNX/EIB Projekten.

Derzeit ist das Programm noch in der Entwicklung.

Aktuelle, übersetzte Snapshots kann man auf dieser
Seite bekommen: http://bugs.freebus.org/projects/show/fts

Die Snapshots funktionieren unter Windows 32/64 bit
und Linux 32/64 bit.

Entpacken, und mit dem Start-Script starten: fts.cmd bzw.
fts.sh, je nach System.

Zum selbst übersetzen aus dem Subversion heraus braucht
man Maven, bzw. das m2eclipse Plugin für Eclipse.
