Freebus Tools Software
----------------------

Ein Programm zur Konfiguration von KNX/EIB Projekten.
Derzeit ist das Programm noch in der Entwicklung.

Aktuelle, übersetzte Snapshots kann man auf dieser
Seite bekommen: https://www.dropbox.com/sh/ujz8cj70chswred/9WHcnQyQmV

Die Snapshots funktionieren unter Windows 32/64 bit und Linux 32/64 bit.

Es muss eine Java 7 Runtime installiert sein. Entweder die von Oracle, oder
die OpenJDK. 

Entpacken, und mit dem Start-Script starten: fts.cmd bzw. fts.sh, je nach
System.

Zum selbst übersetzen aus dem Subversion heraus braucht man Maven, bzw. das
m2eclipse Plugin für Eclipse.

Entwickler, die an FTS arbeiten, sollten die im Sourcecode enthaltene Datei
README_CODING.txt beachten.
